# Прототипы

Статичные HTML/CSS/JS прототипы интерфейсов. Используются как референс при разработке.

| Папка            | Описание                          |
|------------------|-----------------------------------|
| `admin/`         | Прототип панели администратора    |
| `constructor/`   | Прототип конструктора витрин      |
| `showcase-list/` | Прототип списка витрин            |

Для просмотра — открыть `index.html` в браузере.
